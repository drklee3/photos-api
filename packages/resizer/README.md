# Resizer

Resizing service

## Configuration

Environment variables or with an `.env` file.

```env
S3_ENDPOINT=
S3_REGION=
S3_ACCESS_KEY_ID=
S3_SECRET_ACCESS_KEY=
S3_BUCKET=

GRAPHQL_ENDPOINT=http://localhost:4000/graphql
# Generated in prisma seed
GRAPHQL_AUTH_TOKEN=
AMQP_ENDPOINT=amqp://localhost
```
